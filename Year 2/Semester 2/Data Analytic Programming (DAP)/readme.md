# Data Analytic Programming (DAP)

## Reflection

Data Analytic Programming introduced me to the practical side of data analysis using programming techniques. Throughout this course, I learned how to clean, transform, analyze, and visualize data using Python and various data analytics libraries. Working with real datasets helped me understand the complete data analysis workflow, from data preparation to generating meaningful insights.

The projects improved my problem-solving skills and taught me how to use data to support decision-making. I also became more familiar with handling large datasets and presenting findings through visualizations. Overall, this course strengthened my interest in the field of data analytics and provided valuable skills that are highly relevant to my Data Engineering studies.
