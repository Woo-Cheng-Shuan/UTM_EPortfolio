# Operating System (OS)

## Reflection

Operating System provided me with a deeper understanding of how computer systems manage hardware and software resources. I learned about process management, CPU scheduling, memory management, file systems, synchronization, and operating system architecture. These topics helped me understand what happens behind the scenes when applications are executed.

The practical exercises allowed me to explore different scheduling algorithms and system resource management techniques. Through these activities, I developed stronger analytical and problem-solving skills while gaining a greater appreciation for the complexity of modern operating systems. Overall, this course gave me valuable knowledge about the core components that support computer system performance and reliability.
