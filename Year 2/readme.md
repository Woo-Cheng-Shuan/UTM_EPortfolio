# 🚀 Year 2 - Developing Technical Skills

## 📖 Overview

Year 2 was a significant step forward in my academic journey. After building a strong foundation in Year 1, I began exploring more specialized topics related to software development, databases, networking, data analytics, and operating systems. The projects and assignments became more challenging, requiring greater technical skills, teamwork, and problem-solving abilities.

## 📚 Semester 1

### Subjects
- Data Structure & Algorithm (DSA)
- Database
- Network Communication
- Software Engineering

### 🎯 Semester Focus
Understanding how software systems are structured, how data is managed, and how computer networks support communication between systems.

### 💭 Reflection
This semester strengthened my programming and analytical thinking skills. I learned how efficient algorithms and proper database design can improve system performance, while networking and software engineering provided a broader understanding of how large-scale systems are developed and maintained.

---

## 📚 Semester 2

### Subjects
- Data Analytic Programming (DAP)
- Data Mining
- Object Oriented Programming (OOP)
- Operating System (OS)

### 🎯 Semester Focus
Applying programming and analytical techniques to extract insights from data while gaining a deeper understanding of software architecture and system operations.

### 💭 Reflection
This semester introduced me to the world of data analytics and strengthened my interest in data-related fields. I gained hands-on experience working with data, identifying patterns, and building structured software solutions. At the same time, learning about operating systems helped me understand how software interacts with computer hardware behind the scenes.

---

## 🚀 Year 2 Reflection

Year 2 helped me transition from learning fundamental concepts to applying them in more practical and technical scenarios. Through various projects and assignments, I improved my programming, analytical, and teamwork skills while developing a stronger interest in data engineering and data analytics. This year played an important role in shaping my academic direction and preparing me for more advanced subjects in Year 3.
