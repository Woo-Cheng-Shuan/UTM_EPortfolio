# Data Structure & Algorithm (DSA)

## Reflection

Data Structure and Algorithm helped me develop a stronger foundation in problem-solving and computational thinking. Through topics such as arrays, linked lists, stacks, queues, trees, sorting, and searching algorithms, I learned how different data structures affect the efficiency of a program. This course taught me that selecting the right algorithm is just as important as writing correct code.

The programming exercises challenged me to think critically about time and space complexity when designing solutions. Although some topics, especially trees and algorithm analysis, were initially difficult, they improved my logical thinking and coding skills significantly. Overall, this course strengthened my ability to develop efficient and scalable software solutions.
