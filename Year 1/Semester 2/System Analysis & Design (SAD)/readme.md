# System Analysis & Design (SAD)

## Reflection

System Analysis and Design introduced me to the process of developing information systems based on business requirements. I learned how to analyze problems, gather requirements, model business processes, and design system solutions using various diagrams and documentation techniques.

Working on the course project allowed me to experience different phases of the Software Development Life Cycle (SDLC), from requirement analysis to system design. I learned the importance of proper planning, communication, and documentation in ensuring project success. The course also improved my teamwork and presentation skills through group discussions and project development.

If possible, I would like to gain more experience with real industry case studies to better understand how system analysis is performed in professional environments. Overall, this course provided valuable knowledge that bridges business needs and technical solutions.
