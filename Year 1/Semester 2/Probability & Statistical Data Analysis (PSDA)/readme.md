# Probability & Statistical Data Analysis (PSDA)

## Reflection

This course provided me with a strong foundation in probability and statistical analysis, which are essential skills for data-related fields. I learned how to collect, analyze, interpret, and present data using various statistical techniques. Topics such as probability distributions, hypothesis testing, confidence intervals, and regression analysis helped me understand how data can support decision-making.

The practical exercises enhanced my ability to work with data and interpret results critically. I also learned the importance of using appropriate statistical methods to ensure accurate conclusions. These skills are particularly relevant to my studies in Data Engineering, where data-driven decisions play an important role.

To further improve my understanding, I would like to explore more real-world datasets and advanced statistical applications. Overall, this course strengthened my analytical thinking and data interpretation abilities.
