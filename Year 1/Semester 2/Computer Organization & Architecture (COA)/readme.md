# Computer Organization & Architecture (COA)

## Reflection

Computer Organization and Architecture introduced me to the fundamental concepts of how computer systems work internally. Through topics such as CPU architecture, memory hierarchy, instruction execution, and system performance, I gained a deeper understanding of how hardware and software interact. This course helped me appreciate the importance of efficient system design and how hardware decisions can affect overall computing performance.

One of the most valuable lessons I learned was how data is processed at the machine level. Understanding concepts such as registers, cache memory, and instruction cycles gave me a stronger foundation for future computing subjects. The practical exercises also improved my analytical thinking and problem-solving skills when evaluating system performance.

If given the opportunity, I would like to explore more real-world applications of computer architecture, especially modern processor technologies and cloud computing infrastructure. Overall, this course provided a solid foundation that strengthened my understanding of computer systems.
