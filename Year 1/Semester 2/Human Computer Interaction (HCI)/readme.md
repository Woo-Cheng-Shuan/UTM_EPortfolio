# Human Computer Interaction (HCI)

## Reflection

Human Computer Interaction taught me the importance of designing systems that are not only functional but also user-friendly. Through studying usability principles, user experience design, prototyping, and user evaluation techniques, I learned how technology should be developed with users' needs in mind.

The assignments allowed me to apply design thinking and understand how users interact with digital systems. I learned that a successful system is not determined solely by its features but also by how easily users can achieve their goals. This course improved my creativity, communication, and ability to view systems from a user's perspective.

In the future, I would like to gain more experience with advanced UI/UX design tools and user testing methods. Overall, this course helped me develop a user-centered mindset that will be valuable in future software development projects.
