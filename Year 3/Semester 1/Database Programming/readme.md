## 💭 Reflection

This course provided valuable knowledge and hands-on experience in both relational and NoSQL database systems. Through MySQL and MongoDB, an understanding was developed of how different database technologies are designed, managed, and optimized to support various application requirements.

Practical experience was gained in database design, SQL query development, indexing, data manipulation, and NoSQL operations. Through the database project, concepts such as data modeling, performance optimization, and database management were applied in a real-world scenario. Exposure to both structured and semi-structured data also provided a broader perspective on modern database solutions.

Overall, this course strengthened my understanding of data storage and management, which serves as an essential foundation for future studies and projects in Data Engineering.
