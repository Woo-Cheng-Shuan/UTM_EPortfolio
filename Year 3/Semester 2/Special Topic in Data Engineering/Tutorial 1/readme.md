## 💭 Reflection

This project marked my first experience working with Microsoft Azure tools and was found to be both interesting and valuable. At the beginning, difficulties were encountered in understanding the various Azure services and their functions. However, through the tutorial and end-to-end implementation process, a clearer understanding of how different components work together within a complete data pipeline was gradually developed.

Practical exposure was gained in data ingestion, storage, processing, and visualization, providing valuable insights into real-world data engineering practices. Through this experience, a stronger understanding of cloud-based data solutions was developed, along with an appreciation of how modern data pipelines are built and managed.

It is believed that the knowledge and skills gained from this project will be beneficial for future Work-Based Learning (WBL) projects with PPG and will also serve as a strong foundation during industrial training, where cloud technologies such as Microsoft Azure are widely adopted.
