## 💭 Reflection

This tutorial provided valuable exposure to several tools commonly used in modern data engineering workflows, including Apache Spark, PostgreSQL (Docker), Metabase, and Adminer. At the beginning, challenges were encountered while setting up the environment, but they were gradually overcome through troubleshooting and hands-on practice.

Practical experience was gained in using Apache Spark to process data in parallel across multiple cores, demonstrating how large datasets containing millions of records can be handled efficiently. Knowledge of Docker networking and container management was also developed, particularly regarding communication between containers and the importance of configuring persistent storage through volumes.

A deeper understanding of data modelling was acquired through the implementation of a star schema, which highlighted the differences between operational databases and analytical data structures. Experience was also gained in validating data using SQL queries in Adminer and creating visual dashboards through Metabase.

Overall, multiple concepts covered throughout the course, including distributed computing, data modelling, database management, and data visualization, were successfully integrated into a complete end-to-end data engineering pipeline. The knowledge and skills gained from this tutorial have strengthened my understanding of real-world data engineering practices.
