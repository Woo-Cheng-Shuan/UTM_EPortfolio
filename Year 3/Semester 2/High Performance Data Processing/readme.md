## 💭 Reflection

This course provided valuable knowledge on processing large-scale datasets efficiently using modern data processing frameworks. Through hands-on activities and performance analysis, a deeper understanding was developed of how different tools and techniques can be used to improve processing speed, scalability, and resource utilization.

Practical experience was gained in working with large datasets while comparing the performance of technologies such as Pandas, Dask, and Polars. The strengths and limitations of each approach became clearer through experimentation and performance evaluation. Concepts related to parallel processing, distributed computing, and big data analytics were also explored throughout the course.

Overall, this course strengthened my technical understanding of large-scale data processing and provided valuable skills that are highly relevant to the field of Data Engineering, where efficiency and scalability are critical requirements.
